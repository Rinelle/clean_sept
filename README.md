# Структура проекта:
* В названии файлов **scss** которые не **main.scss** нужно ставить нижнее подчеркивание: **_base.scss**
* Шрифты подключать не нужно. Они уже подключены.

 ```
├── assets
|   ├── images
|   └── scss
|       ├── base
|       |   ├── base.scss
|       |   ├── fonts.scss
|       |   ├── mixins.scss
|       |   ├── reset.scss
|       |   └── vars.scss
|       ├── blocks
|       └── main.scss
├── .gitignore
└── index.html
 ```
